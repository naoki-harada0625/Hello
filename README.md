# Hello World HTML Project

This project is a simple HTML application that displays "Hello World" on a web page. It serves as an introductory example for those learning HTML.

## Project Structure

```
hello-world-html
├── src
│   └── index.html
└── README.md
```

## Files

- `src/index.html`: This is the entry point of the HTML page. It contains the basic structure of an HTML document and uses an `<h1>` tag to display "Hello World".

## Usage

To view the "Hello World" page, open the `src/index.html` file in a web browser. You should see the message displayed prominently on the page. This project is intended for educational purposes to demonstrate basic HTML structure and elements.
